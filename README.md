### Installation

1. Clone the repository to your local machine:

```bash
git clone https://github.com/HheZi/SpringBootChatApplication
```

2. Navigate to the project directory:

```bash
cd Bookstore
```

3. Compile the source code:

```bash
javac -d bin src/*.java
```
